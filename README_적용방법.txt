[전체 슬라이드 스캔 버전]

수정 내용
1. 스토리보드 PPT의 모든 슬라이드를 1장부터 마지막 장까지 매칭 후보로 사용합니다.
2. 설계가이드/과정정보/표지/구분 화면이 메타로 감지되어도 제외하지 않습니다.
3. Vision OCR 보강도 메타 감지 여부와 무관하게 전체 슬라이드에 적용됩니다.
4. 모션 종료 프레임/스토리보드 앵커/STEP 로고 검출 기능은 유지합니다.

적용 방법
- 기존 core_video.py, app_video.py를 이 폴더의 파일로 교체하세요.
- STEP 로고 파일은 app_video.py와 같은 폴더에 두세요.
- 실행: streamlit run app_video.py
